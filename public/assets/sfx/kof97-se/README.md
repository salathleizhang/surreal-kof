Auto-sliced (on silence) from `42.S.E. Collection.flac` of the KOF '97 OST pack.
166 clips: mostly impact / hit / move SFX, plus some system/announcer sounds.

  original 288s collection, in seconds), for identifying what each one is.

Saved for later use (hit/impact SFX in the fight scene). Not wired into the game yet.
